# API da Programacao TDC Floripa 2026

API simples para a Custom Tool do OCI Generative AI Agent.

## Rodar localmente

```bash
npm start
```

Por padrao, a API sobe em:

```text
http://localhost:3000
```

## Endpoints

```text
GET /event
GET /tracks?day=22/jul
GET /sessions?q=agentes&limit=5
GET /sessions?speaker=Ana%20Lindiner
GET /speakers?q=ana
```
